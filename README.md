# Script

## วิธีใช้งาน

เข้าไปที่ directory Script

```bash
  cd Script
```

จากนั้นก็สั่งรัน Script ด้วยคำสั่ง 


```bash
  ./ชื่อScriptที่ต้องการ
```

รอ รอ และ รอ เสร็จ

ต่อไปจะเป็นการ Config อื่นๆที่ยังไม่ได้ทำ
